Features

Designed for small tasks, interruptions, and imperfect memory.

Quick Capture

Write things down first without forcing yourself to organize every detail immediately.

Completion Comments

Add multiple follow-up notes after a task is done, so results and context are not lost.

Priority Quadrants

Use important urgent, important not urgent, not important urgent, and not important not urgent filters.

History and Reports

Keep creation time, completion time, and comments for weekly or monthly review.

Desktop

Open it like a regular Windows app, no browser needed.

The desktop app is built with Electron and supports Windows 10. The app name and local data directory are configurable, so the project can be reused by other people after open sourcing.

Local First

Your task data stays on your computer by default.

Task data is stored at Documents/DaheTodo/tasks.json by default, and the directory can be changed in settings. The repository includes privacy checks to avoid committing local task data, installers, caches, and tool directories.

Get Started

Download, install, and start capturing tasks.

  1. Download the Windows installer from GitHub Releases.
  2. Open the desktop app and confirm the data storage location in settings.
  3. Add tasks, append comments after completion, and collect report material from the report page.
