本地优先 · 快速收集 · 番茄钟 · 每周成就

把突然插进来的琐事，稳稳接住。

大何的待办事项是一个面向办公场景的 Windows 桌面待办工具：随手记录、专注番茄、完成备注、按周复盘，并把过程沉淀成可复制的周报素材。

v0.5.0 亮点

从“记录事情”升级到“看见本周投入”。

Focus

任务绑定番茄钟

给当前事项开启专注，记录完整番茄、中断原因、累计时间和今日投入。

Achievement

每周成就系统

按周自动解锁“本周起步”“稳住节奏”“闭环大师”等成就，为复盘提供正反馈。

Report

周报素材增强

周报自动汇总完成事项、评论、专注时长、中断次数和本期成就。

能力矩阵

不是大而全，而是围绕“容易被打断的人”做完整闭环。

捕获

快速录入事项、补充信息、分组和优先级，先写下来，避免靠脑子硬记。

执行

四象限优先级筛选，专注番茄约束当前任务，插入事项自动进入重要紧急。

沉淀

完成后可追加多条评论，保留创建时间、办理时间、专注记录和中断记录。

复盘

历史按周/月分类，周报生成素材，成就系统把本周投入变成可见反馈。

数据与隐私

默认不联网，不上传，不需要账号。

桌面端默认写入 %USERPROFILE%\Documents\DaheTodo，其中 `tasks.json` 保存待办，`focus-sessions.json` 保存专注记录，`achievements.json` 保存成就解锁记录。

仓库内置隐私检查和 pre-push 钩子，避免本地任务数据、安装包、依赖缓存和工具目录被误提交到 GitHub。
