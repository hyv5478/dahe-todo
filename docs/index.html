把突然插进来的琐事，稳稳接住。

一个轻量的本地待办工具：随手记录、完成打钩、追加评论、按时间和优先级回看，最后把完成过程整理成周报素材。

核心功能

为琐碎、插队、容易忘的工作流设计。

快速收集

事情冒出来时先记下来，不要求你立刻想清楚结构，降低打断当前工作的成本。

完成后追加评论

办完后可以多次补充过程、结果、原因和影响，后续整理周报时不用重新翻记忆。

优先级四象限

支持重要紧急、重要不紧急、不重要紧急、不重要不紧急，并能在待办页直接筛选。

历史和周报

保留创建时间、完成时间和评论记录，按周或月回看，快速形成周报素材。

桌面端

不用浏览器，也能像普通软件一样打开。

桌面版基于 Electron 构建，支持安装到 Windows 10。本地数据位置可配置，应用名称也可配置，后续开源给其他人使用时不绑定个人名字。

本地优先

数据默认留在你的电脑里。

待办数据默认写入 Documents/DaheTodo/tasks.json，也可以在设置里更换目录。仓库已经加入隐私检查，避免把本地任务数据、安装包、缓存和工具目录提交到 GitHub。

English

Local-first todo app for interruption-heavy work.

A lightweight Windows desktop todo app for quick capture, completion notes, priority quadrants, history review, and weekly-report material collection. Task data stays local by default.

开始使用

下载、安装，然后直接记录。

  1. 进入 GitHub Releases 下载 Windows 安装包。
  2. 安装后打开桌面版，先在设置里确认数据存储位置。
  3. 新增待办，完成后追加评论，周报页会帮你汇总素材。
