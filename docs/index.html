大何的待办事项
功能 桌面端 本地存储 开始使用

Windows 本地优先 · 桌面待办 · 周报素材

把突然插进来的琐事，稳稳接住。

一个给高频事务场景做的待办工具：快速记录、打钩完成、追加过程评论、按周/月回看，并把完成记录整理成周报素材。

下载安装包 查看 GitHub

Features

为真实工作流设计

随手记录

事情一来先落地，不靠脑子硬记。支持补充说明、分组和优先级。

完成评论

一件事可以追加多条评论，保留时间线，也能修改旧评论。

全局搜索

标题、说明、分组、评论都能搜，事项多了也能快速翻出来。

周报素材

按日期整理已完成事项和评论，直接复制成 Markdown。

Desktop

不是浏览器标签页，是独立桌面软件。

桌面端使用 Electron 构建，支持安装包、桌面快捷方式、可配置项目名称和数据保存位置。浏览器版仍保留，方便开发和轻量使用。

Local First

数据在你电脑上。

默认保存到 Documents/DaheTodo/tasks.json，也可以在设置里改位置。应用核心功能不需要联网，不上传待办和评论。

Start

开源发布准备好了。

  1. 在 GitHub Releases 上传 Windows 安装包。
  2. 启用 GitHub Pages，目录选择 docs/。
  3. 把页面里的 GitHub 链接替换成你的真实仓库地址。
